# kotlin-playground

A Kotlin playground project for experimenting and learning.

## Stack

- Kotlin 2.1.20
- Java 23
- Gradle (Kotlin DSL) with version catalogs
- [Kotest](https://kotest.io/) for testing
- [kotlin-logging](https://github.com/oshai/kotlin-logging) + Logback

## Build & Run

```bash
./gradlew build
./gradlew test
```
